# AutoHeroicQueue

WoW-аддон для сервера **Sirus**: автоматически выставляет сохранённый тип
очереди в дропдауне "Тип" окна Dungeon Finder при входе в игру и при каждом
открытии окна, чтобы не выбирать его руками каждый раз.

## Возможности

- Запоминает выбранный тип очереди (например героик/испытание/конкретное
  подземелье) и подставляет его автоматически:
  - через ~3 секунды после входа в игру (`PLAYER_ENTERING_WORLD`);
  - при каждом открытии окна LFD (`LFDParentFrame` → `OnShow`).
- Команда `/ahq` открывает окно выбора: список всех типов, доступных на
  текущем уровне персонажа, с подсветкой текущего выбора.
- Поддержка скина ElvUI (если он установлен и загружен) с фолбэком на
  стандартный Blizzard-фон, если ElvUI не найден.

## Установка

1. Скачай репозиторий (Code → Download ZIP, либо `git clone`).
2. Положи папку `AutoHeroicQueue` (с файлами `AutoHeroicQueue.toc` и
   `AutoHeroicQueue.lua`) в:
   ```
   <папка игры>/Interface/AddOns/AutoHeroicQueue/
   ```
3. Перезапусти игру или сделай `/reload`.

## Использование

- `/ahq` — открыть окно выбора подземелья/типа очереди. Выбор сохраняется
  навсегда (через `SavedVariables`) до следующего изменения.
- После выбора аддон сразу пробует применить его к текущему открытому окну
  LFD, если оно открыто.

## Технические детали

Аддон использует стандартные близзардовские API `LFDQueueFrame`,
`LFDQueueFrame_SetType` и `LFDParentFrame` — то есть завязан на "родной"
интерфейс Dungeon Finder, без переопределений сторонних UI-фреймворков.
Список доступных типов собирается через `GetNumRandomDungeons` /
`GetLFGRandomDungeonInfo` / `GetLFGDungeonInfo` с теми же условиями
фильтрации по уровню и стадии дополнения, что использует сама Blizzard.

> Обрати внимание: эта версия рассчитана именно на Sirus (стандартный
> `LFDQueueFrame`). Для серверов с кастомным UI Dungeon Finder (например
> Ascension, где окно и дропдаун реализованы через собственные фреймы)
> нужна отдельная адаптация — см. ветку/репозиторий `AutoHeroicQueue-Ascension`.

## Отладка

По умолчанию включён debug-вывод в чат (`DEBUG = true` в начале файла).
Чтобы отключить, поставь `DEBUG = false`.
